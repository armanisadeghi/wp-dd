== Description ==

The Fast Page & Post Duplicator plugin lets you make a copy of any post or page in WordPress with one click.

It makes a copy of the original content so you can use the same layouts, formatting and structure again without having to start from scratch.

This plugin is simple and fast. It makes managing content easier.

It is great for:

* Using the same page layouts
* Making versions of landing pages
* Copying posts to update or translate them
* Making copies of templates quickly

== Features ==

✔ Duplicate posts with one click
✔ Duplicate pages with one click
✔ It copies the content, title and settings
✔ Keeps the structure and formatting as the original post
✔ Fast and simple
✔ With a simple interface in the WordPress post list

New things in recent versions:

✔ See what you have cloned recently
✔ View statistics on who is cloning what
✔ Check history of all the clones you have made

== How it works ==

1. Install and activate the Fast Page & Post Duplicator plugin.
2. Go to **Posts → All Posts** or **Pages → All Pages**.
3. Move your mouse over any item in the list.
4. Click **Clone**.
5. The copied content will open as a **draft**. You can edit and publish it.

== Why use Fast Page & Post Duplicator? ==

Making pages from scratch can take a time.

With the Fast Page & Post Duplicator plugin you can make a copy of existing content. Change it which saves you time and makes sure your site looks consistent.

The Fast Page & Post Duplicator plugin works with the WordPress admin interface and does not need any special setup.

== Installation ==

1. Upload the Fast Page & Post Duplicator plugin to `/wp-content/plugins/` or install it from the add plugin from your backoffice
2. Activate it through the **Plugins** menu in WordPress
3. Go to **Posts** or **Pages**
4. Move your mouse over any item. Click **Clone**

== Frequently Asked Questions ==

= Does the Fast Page & Post Duplicator plugin duplicate both posts and pages? =

Yes. The Fast Page & Post Duplicator plugin lets you make a copy of both posts and pages from the WordPress list view.

= Where is the copy created? =

The copied content is created as a **draft** so you can safely edit it before you publish it.

= Does it keep the formatting as the original? =

Yes. The Fast Page & Post Duplicator plugin keeps the content and formatting of the post or page.
